### How to Install ?

Clone 

``` sh

git clone https://github.com/julles/maxima-tes.git

```

Copy .env.example file to .env

``` sh 

cp .env.example .env

```

Install Composer depedencies

``` sh

composer install

```

Migrating Database

``` sh 
php artisan migrate

``` 

Finish
